# 关于颜色
因为项目含多主题，所以在使用颜色时候必须到指定位置查看

## 目前的主题
- defailt   /src/uni.scss
- dark      /src/styles/base/theme/var/dark
后续的拓展都会继续在这里增加 `/src/styles/base/theme/var`
